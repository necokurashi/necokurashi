Itadaki Japan Poster

Travel videos about Japan, published to our own TikTok — automatically.

Itadaki Japan Poster is an internal automation tool operated by Itadaki Japan. It publishes our own short travel videos about Japan to our own TikTok creator account, @itadaki.japan, on a set schedule.

What it does

Each day our production pipeline creates a short travel video. The tool signs in to our own TikTok account using TikTok Login, then publishes the finished video directly to that account through the TikTok Content Posting API.

TikTok capabilities used

Login Kit — user.info.basic Content Posting API — video.publish Content Posting API — video.upload

We access only our own account's basic profile information to confirm the correct account is connected, and we use the Content Posting API (Direct Post) to publish our own videos. The tool is not offered to the public.
